MIRA Sodexo, 2014

Desktop application that automates rudimentary research and consolidation of user-submitted “Missing Invoice” requests from customers. Used by corporate-level Accounts Payable staff.

Features:

  • Ingests and parses ‘Missing Invoice’ survey responses from end-users .
  • Integrates with corporate Oracle transaction database to execute SQL queries.
  • Automates basic logic for identifying common root-causes of missing invoices.
  • Outputs list of inquiries with current transaction status and root-cause.
