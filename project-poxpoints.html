PoxPoints PoxPoints, 2015

.NET MVC 4 Web Application for users of an online game to buy, sell, and trade in-game virtual items for real cash via PayPal and a custom-built game API.

Features:

  • Custom-built transaction system for buy-sell-trade of virtual items including automated pricing of goods via custom retail margins and item-scarcity.
  • Custom-built API for manipulating Poxnora.com’s user authentication and trading interface reverse-engineered using Fiddler Web Debugging Proxy.
  • Online Entity Framework database of inventory, prices, item details, and transaction records with extensive use of LINQ. Ingests game’s JSON feed of item details into database.
  • Implements PayPal’s Payments REST API for cash transaction.
  • Dynamic Razor HTML front-end.
  • Deployment via MS Azure, MS Team Foundation Source Control.
